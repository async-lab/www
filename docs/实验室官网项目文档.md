# 实验室官网项目文档

## 项目概述

**项目名称**：开发实验室官网  
**实验室定位**：全栈开发实验室（前端、后端、移动端、嵌入式）  
**项目类型**：纯前端静态网站  
**设计风格**：科技公司级别的高级感 + 简约大气 + 流畅高级动画  
**目标**：展示实验室介绍、研究方向、团队成员、科研成果、新闻动态等信息  
**技术栈**：Vue 3 + Vite + TypeScript + Vue Router + Pinia + GSAP + Tailwind CSS

## 功能模块

### 1. 首页 (Home)
- **Hero Section**：
  - 全屏沉浸式背景（满屏代码效果，单词随机变换，Advance、Synergy、Yield、Nexus、Create）
  - 进入屏幕蓝白线性渐变条纹带划过后停留，占画面右半1/3.
  - 实验室名称大标题，打字机效果逐字显示
  - Slogan logo淡入动画
  - 视差滚动效果
- **研究方向模块**（四大方向卡片）：
  - 前端开发（Frontend）：图标 + 技术栈标签（Vue/Nuxt/小程序）
  - 后端开发（Backend）：图标 + 技术栈标签（Node.js/Java/Python）
  - 移动端开发（Mobile）：图标 + 技术栈标签（uni-app/Flutter/原生）
  - 嵌入式开发（Embedded）：图标 + 技术栈标签（ARM/STM32/物联网）
  - 四个卡片在滚动时向后翻，随后下一个卡片从上翻下来，整体页面不滚动，只有卡片翻转。

### 2. 关于我们 (About)
- **实验室历史**：时间轴可视化展示，滚动时触发横向滚动显示
- **研究方向详解**：
  - **前端开发**：Web 应用、跨平台小程序、性能优化、工程化
  - **后端开发**：微服务架构、分布式系统、数据库优化、DevOps
  - **移动端开发**：原生/跨平台应用、性能调优、用户体验设计
  - **嵌入式开发**：物联网、智能硬件、实时系统、边缘计算
  - 每个方向配合图标和技术栈标签云
- **实验室文化**：代码即艺术、技术驱动创新

### 3. 团队成员 (Team) --暂不需要
- **导师介绍**：
  - 大头像卡片，Hover 时展示详细信息（研究方向、联系方式、个人主页）

  - 按方向分组（前端组、后端组、移动端组、嵌入式组）

### 4. 研究成果 (Publications / Projects) --暂不需要

  - 卡片式布局，支持按方向筛选（前端/后端/移动端/嵌入式）
  - 每个项目卡片包含：

- **技术博客**：文章列表，支持标签筛选
- **开源贡献**：GitHub 贡献图可视化展示

### 5. 新闻动态 (News) --暂不需要

## 技术实现方案（科技公司级别）

### 核心技术栈

**前端框架**：Vue 3 + Vite
- Vue 3 Composition API 提升组件组织和复用能力
- Vite 提供极速开发服务器和高性能生产构建
- 可按需扩展 Nuxt 3，用于 SSR/SSG 和更强 SEO 场景

**开发语言**：TypeScript
- 类型安全，减少运行时错误
- 更好的代码提示和维护性

**样式方案**：Tailwind CSS
- 原子化 CSS，快速构建 UI
- 内置响应式和深色模式支持
- 配合 `class-variance-authority` 实现组件样式管理

**动画库**（核心）：
- **GSAP**：生产级动画库
  - 页面切换动画
  - 滚动触发动画（ScrollTrigger）
  - 复杂时间轴动画
  - 组件进入/退出动画
- **Vue Transition / TransitionGroup**：轻量状态过渡
  - 列表过渡
  - 弹窗、菜单、路由基础过渡
- **Three.js**（可选）：
  - Hero 区域粒子系统
  - 3D 变换动画
  - 3D 背景效果和交互式模型展示

**UI 组件库**：

- **Element Plus**：成熟稳定的 Vue 3 组件库，适合后台、列表、表单等高效开发
- **Headless UI Vue / Reka UI**：无样式的可访问组件原语，适合自定义高级视觉
- **VueUse**：常用组合式函数集合，提升交互开发效率
- 自定义高级动画组件

**图标库**：
- **Lucide Vue Next**：现代简洁图标
- **Iconify / unplugin-icons**：按需引入多图标集

**数据可视化**：
- **ECharts / vue-echarts**：图表库（技术栈雷达图、数据统计）
- **D3.js**（可选）：复杂可视化

**其他工具**：
- **Vue Router**：页面路由和路由过渡
- **Pinia**：轻量状态管理
- **@vueuse/core**：视口检测、媒体查询、深色模式等常用能力
- **vite-plugin-pages**（可选）：文件系统路由
- **vite-plugin-vue-layouts**（可选）：布局系统

### 数据管理
- **静态数据**：JSON 或 Markdown 文件存储（成员、项目、新闻等）
- **内容管理**：支持 Markdown / MDX-like 内容方案，或使用 VitePress 风格的内容组织
- **图片优化**：Vite 插件、现代图片格式和懒加载策略

### 性能优化策略
- **代码分割**：Vue Router 路由级懒加载 + Vite 自动分包
- **图片懒加载**：Intersection Observer + 原生 `loading="lazy"` + 图片压缩
- **动画性能**：优先使用 CSS transform 和 opacity，避免重排
- **字体优化**：本地字体托管 + `font-display: swap`
- **预加载关键资源**：Link prefetch

### 部署方案
- **推荐**：Netlify / Cloudflare Pages / Vercel（均支持 Vite 静态站点自动 CI/CD）
- **备选**：GitHub Pages / 静态资源托管
- **自建**：Docker + Nginx

## 设计风格指南（科技公司级别）

### 色彩系统

**主色调**：科技蓝 + 深空灰
```css
/* 品牌色 */
--primary: #0066FF;        /* 科技蓝 */
--primary-dark: #0052CC;   /* 深蓝 */
--primary-light: #3385FF;  /* 浅蓝 */

/* 辅助色（四大方向） */
--frontend: #00D4FF;       /* 青蓝 - 前端 */
--backend: #00FF88;        /* 翠绿 - 后端 */
--mobile: #FF00FF;         /* 品红 - 移动端 */
--embedded: #FFB800;       /* 琥珀 - 嵌入式 */

/* 中性色 */
--gray-50: #F9FAFB;
--gray-900: #111827;
--background: #0A0A0A;     /* 深色背景 */
--surface: #1A1A1A;        /* 卡片背景 */
```

**深色模式优先**：默认深色，可切换浅色

### 排版系统

**字体族**：
- **中文**：`"HarmonyOS Sans", "PingFang SC", "Microsoft YaHei"`
- **英文/代码**：`"Inter", "SF Pro Display", "Roboto"`
- **等宽字体**：`"Fira Code", "JetBrains Mono"`（用于技术栈标签）

**字体大小**：
```css
--text-xs: 0.75rem;    /* 12px - 标签 */
--text-sm: 0.875rem;   /* 14px - 辅助文字 */
--text-base: 1rem;     /* 16px - 正文 */
--text-lg: 1.125rem;   /* 18px - 小标题 */
--text-xl: 1.25rem;    /* 20px */
--text-2xl: 1.5rem;    /* 24px */
--text-3xl: 1.875rem;  /* 30px */
--text-4xl: 2.25rem;   /* 36px - 大标题 */
--text-5xl: 3rem;      /* 48px - Hero 标题 */
```

### 布局原则

- **留白充足**：元素间距至少 16px，大区块间距 80-120px
- **最大宽度**：内容区 1200px - 1400px
- **网格系统**：12 列响应式网格
- **卡片圆角**：8px - 16px（大卡片用 16px）
- **阴影层次**：
  ```css
  --shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
  --shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.1);
  --shadow-xl: 0 20px 25px rgba(0, 0, 0, 0.15);
  --shadow-glow: 0 0 20px rgba(0, 102, 255, 0.3); /* 发光效果 */
  ```

### 动画原则

**缓动函数**：
```css
--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);      /* 爆发式减速 */
--ease-in-out-cubic: cubic-bezier(0.65, 0, 0.35, 1); /* 平滑进出 */
--ease-spring: cubic-bezier(0.68, -0.55, 0.265, 1.55); /* 弹簧效果 */
```

**动画时长**：
- 微交互：150ms - 250ms（按钮、Hover）
- 中等动画：350ms - 500ms（卡片翻转、模态框）
- 大型动画：600ms - 1000ms（页面切换、Hero 动画）

**动画类型**：
- **Hover 效果**：transform: scale(1.05) + 阴影增强
- **滚动触发**：opacity: 0 → 1 + translateY(50px) → 0
- **页面切换**：淡入淡出 + 轻微位移
- **加载动画**：骨架屏 + 脉冲效果

### 视觉特效

**Glassmorphism（毛玻璃）**：
```css
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(10px);
border: 1px solid rgba(255, 255, 255, 0.1);
```

**渐变背景**：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
background: radial-gradient(ellipse at top, #0066FF, transparent);
```

**光晕效果**：
```css
box-shadow: 0 0 60px rgba(0, 102, 255, 0.4);
```

### 组件设计规范

**按钮**：
- Primary：蓝色填充 + 白色文字 + Hover 时发光
- Secondary：透明背景 + 蓝色边框 + Hover 时填充
- Ghost：仅文字 + Hover 时背景淡入

**卡片**：
- 深色背景 + 细边框
- Hover 时浮起（translateY: -8px）+ 阴影增强
- 内边距：24px - 32px

**技术栈标签**：
- 小药丸形状（border-radius: 9999px）
- 半透明背景 + 方向色边框
- 字体：等宽字体（Fira Code）

**导航栏**：
- 毛玻璃效果 + 固定顶部
- 滚动时背景透明度变化
- Logo + 导航链接 + 深色模式切换

## 注意事项

1. **性能优先**：
   - 首屏加载时间 < 2s
   - Lighthouse 性能评分 > 90
   - 动画帧率稳定在 60fps
   - 使用 Vue 异步组件和路由懒加载非关键页面

2. **动画性能优化**：
   - 优先使用 transform 和 opacity（GPU 加速）
   - 避免动画过程中触发 layout 和 paint
   - 使用 will-change 提示浏览器优化
   - 复杂动画使用 Canvas/WebGL

3. **可访问性**（A11y）：
   - 语义化 HTML 标签
   - ARIA 标签完善
   - 键盘导航支持
   - 屏幕阅读器友好
   - 色彩对比度符合 WCAG 2.1 AA 标准

4. **浏览器兼容性**：
   - 现代浏览器（Chrome/Firefox/Safari/Edge 最新版）
   - 优雅降级：动画在低性能设备上简化

5. **内容管理**：
   - JSON 数据驱动，方便更新
   - 图片统一命名规范
   - 代码注释完善

6. **SEO 优化**：
   - 每个页面独立 meta 标签
   - 生成 sitemap.xml
   - 使用语义化标签
   - Open Graph 和 Twitter Card 支持

7. **隐私和版权**：
   - 成员信息需授权
   - 图片版权合规
   - 开源协议遵守
