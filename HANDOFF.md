- 新增 `Why Async` 说明块，用计算机异步处理的概念解释实验室名称与成长理念。

### 2. 信息结构与视觉层级完善

- 品牌缩写由 `DL` 统一为 `AL`。

- 页面标题、Open Graph 标题和描述更新为“异步开发实验室官网”。

- GitHub 联系项改为可访问链接，并沿用项目现有的 `https://github.com/async-lab`。

### 4. 锚点导航修复

视觉检查时发现，直接访问 `/about` 或 `/#about` 会受到 ScrollSmoother、ScrollTrigger 与 hash 定位时序影响：

- 桌面端可能已滚到介绍区，但 `ScrollReveal` 内容仍保持隐藏。
- 手机端可能把 `01 / About` 定位到固定导航下面。

`src/App.vue` 现已在锚点定位后重新刷新 ScrollTrigger，并在无 ScrollSmoother 的紧凑视口中使用明确的 96px 顶部偏移。`/about` 仍会兼容跳转到 `/#about`。
